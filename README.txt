------ DEPENDENCIAS ------
pip install PyOpenGL PyOpenGL-accelerate
pip install glfw
pip install numpy
python 3.11>

------ POSSIVEIS ERROS ------
Se der erro de permissão pause o OneDrive e confira se o pontos.txt esta com escrita.
Confira se o caminho do console está na mesma pasta do arquivo .py
Recomendado o uso do VSCode

------ SOBRE ------
O Projeto é sobre um simulador fictício de desenho, onde a comunicação é duplex feita
com o protocolo TCP/IP.

------ COMO USAR ------
Clique P para simular o recebimento de dados (com o foco na tela do openGL).
Clique em 2 pontos da janela para simular o envio de dados.
